# Briefing review

Use this skill when you need to critique a draft before producing the final version.

## Review criteria

1. Clarity: the main idea is obvious in the first few lines.
2. Credibility: claims are grounded and caveats are visible.
3. Demo value: the demo proves a concept instead of only showing output.
4. Audience fit: the level of detail matches a live presentation.
5. Actionability: the final version gives the presenter a clear next line.

## Output shape

Return:

- Verdict
- What is working
- What to change
- Final rewrite guidance

Use `read_skill_resource` for `references/review-rubric.md` when you need the scoring rubric.
